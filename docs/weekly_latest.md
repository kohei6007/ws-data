## 今週の教訓（4182 取引からの学び）

| # | 教訓 |
|---|---|
| 1 | 5MA陰線下抜けの翌日寄付き売却は**絶対**。寄付き売却で底値を引いても、後日のさらなる下落リスクと比較すれば必ず合理的 |
| 2 | 過去3-4ヶ月前の出来高最大日も上値抵抗として認識（4186・4182 共通） |
| 3 | 同セクター先行サインを察知（4062・5803 の前日動向） |
| 4 | キンカブ寄付き売買では予兆段階での売り予約準備が重要 |
| 5 | 損切ライン -5%（マニュアル基準）と -6〜-7%（実運用）の両方を併記 |
| 6 | 「中長期 MA 上向き」を理由に保有継続しない |
| 7 | 「もう一段反発するかも」の心理を排する |
| 8 | 全体相場が良くても自セクターが悪ければ巻き添えで売られる |
