# 取引から得た教訓集

過去の取引から見えてきたパターン化された学び。同じ失敗を繰り返さないために。

## 教訓 #1：5MA陰線下抜けの翌日は売り

- **根拠**：マニュアル §9 3次「5MA横ばい or 下向きを陰線で抜ける → 売却」
- **実例**：
  - 4186 東京応化（5/27 下抜け → 5/29 前場売却・-1.80%）→ 売却なしなら損切ライン到達していた（-5%）
- **応用**：損切ラインに余裕があっても、§9 シグナルが出たら即執行が安全
- **アンチパターン**：「中長期MAは上向きだから保有」と粘ると、結局損切まで到達するリスク高い

## 教訓 #2：「上がるか下がるかは誰にもわからない」を尊重

- **根拠**：マニュアル §9 2次の明示的記述
- **意味**：自分の予測（戻るはず）に固執せず、相場のシグナルに従う
- **適用**：5MA下抜け後の「戻りを信じて待つ」は禁物

## 教訓 #3：未話題ゾーンは連れ高に乗りやすいが、主役が崩れると先に崩れる

- **観察**：CNBC で話題にならない半導体材料系（4186, 4182）は、主役（東エレ等）が好調な時は連れ高するが、主役が崩れると先に売られやすい
- **理由**：流動性が低く、利確売りで深く下げやすい
- **対策**：未話題ゾーンの保有銘柄こそ、5MA下抜けで早めに撤退する判断が重要

## 教訓 #4：パーフェクトオーダーは万能ではない

- **観察**：4186 / 4182 ともに月足・週足・日足のパーフェクトオーダー成立でエントリー → 短期で 5MA陰線下抜け
- **意味**：中長期トレンドが健全でも、短期の調整は深い場合がある
- **対策**：パーフェクトオーダー＝買い、ではなく「+ 5MA への引きつけ + 陽線」が揃って初めて買い

## 教訓 #5：データ反映の時差に注意

- **観察**：yfinance は当日終値の反映に遅延がある（前回も問題化）
- **対策**：kabutan の確定値で補完するロジックを analyze.py に実装済（v3）
- **判断時**：「最新バー」の日付を必ず確認

## 教訓 #6：Volume Surge Pivot（VSP）を判定軸に加える

- **概念**：直近で出来高が急増した日の OHLC 中央値より「少し下」を基準価格に設定
- **使い方**：
  - 現在値 > 基準 → 上昇トレンド継続バイアス
  - 現在値 ≤ 基準 → 撤退シグナル
- **4186 実例**：5/11-12 出来高ピーク → 中央値 11,422 → 基準 10,900円
  - 5/28 終値 10,900 で基準到達 → §9 と整合した撤退シグナル
- **詳細**：`references/volume-surge-pivot.md`
- **既存スキルとの併用**：§9（5MA下抜け）× VSP × 直近高値未突破 = 3つ揃ったら機械的撤退

## 教訓 #7：直近高値の上値抵抗を意識（短期）

- **観察**：陰線で確定した直近高値は、次の上昇局面の上値目安として機能
- **4186 実例**：5/15 直近高値 11,860 → 5/25 高値 11,815 で突破できず → 上昇継続力の陰り
- **応用**：エントリー後、直近高値を超えられない日が出たら警戒する

## 教訓 #7.5：過去3-4ヶ月前の出来高最大日も上値抵抗として認識する ★重要

- **観察**：1-2ヶ月の直近高値だけでなく、**3-4ヶ月前の出来高最大日**の高値も重要な節目
- **4182 実例（6/4 時点）**：
  - 直近高値（5/29 高値 5,626）だけを上値として認識していた
  - **3/3 が出来高最大（7,744,600株）で高値 5,376**を見落としていた
  - 結果的に「5/29 5,626 と 3/3 5,376 両方の上値より下＋陰線」の3条件揃いで撤退判断が確定
- **対策**：VSP 計算は **過去3-4ヶ月遡る**こと（30-60日では不足）
- **撤退条件強化版**：
  1. マニュアル §9 5MA陰線下抜け
  2. 直近高値（1-2ヶ月内）を未突破
  3. **過去3-4ヶ月の出来高最大日の高値を未突破**
  4. 当日が陰線
  → 4条件揃ったら確定的な撤退シグナル

## 教訓 #8.5：キンカブ・寄付き売買のみ前提の取引設計 ★最重要

- **大前提**：取引商品はキンカブ（金額指定単元株未満取引）。**売買は寄付き成行のみ**
- **判断は前夜〜翌朝寄付き前までに完了必須**（場中ははりつかない）
- **寄付き売却は底値を引きやすい**（売り殺到で）
  - 4182 実例：寄付 4,999 / 終値 5,180 = +181円戻し（寄付が底値）
  - 損切ライン -5% でも実損 -7% 超もあり得る
- **シグナル発動を「予兆段階」で察知する仕組みが重要**
  - 5MA との距離 +1% 以内 → 警戒ライン
  - 同セクター仲間下落 → セクター全体警戒
  - 米半導体時間外 -3% 以上 → 半導体材料系警戒
- **シグナル発動確認 → 翌朝寄付き売り予約は必須行動**（次善策ではない）
- **4182 実例**：6/4 大引けで売却シグナル確認 → 6/5 寄付 -7% で売却。6/3 夜の予兆段階で予約していれば 6/4 寄付 5,400 付近で売れた可能性（+400円改善）

## 教訓 #8：フライングエントリーの心理を排する

- **観察**：AI過熱期・他銘柄急騰時、「乗り遅れる」感情でエントリーを1日早めることがある
- **4186 実例**：本来 5/26 寄付きエントリーが正解 → 心理的に 5/25 寄付きにフライング
- **結果検証**：今回は 5/25 終値が想定通り上昇したので大事には至らず
- **教訓**：取れる利益は微小（5円程度）、ルール遵守の方が長期的に安全
- **対策**：「他銘柄が上がってる = 自分の銘柄も明日上がる」とは限らない、エントリー条件は厳密に守る
