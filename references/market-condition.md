# 全体相場（地合い）の判定

出典：投資マニュアル §10〜§12

## 地合いの3区分（日経平均・S&P500 の日足MA向きで判定）

| パターン | 上がる材料 | 下落する材料 | 指数の日足MA |
|---|---|---|---|
| **イージー**（勝率高い） | ある | ない（ほとんど） | 5MA・10MA・20MA が右肩上がり |
| **ノーマル**（普通・わからない） | ある | 少しある | 5MA・10MA・20MA が横ばい |
| **ハード**（勝率低い） | ほとんどない | ある | 5MA・10MA・20MA が右肩下がり |

> 簡易判定：**日足20MAが右肩上がりであればとりあえずは地合いは良いほう**（§14 Q&A）

## 上がる材料 / 下落する材料

| 材料 | 上がる | 下落する |
|---|---|---|
| 景気の先行き（中長期：米雇用統計・GDP・小売売上高・長期金利） | 良い | 悪い |
| 金融政策（FOMC・日銀） | 利下げ・BSの拡大 | 利上げ・BSの縮小 |
| 物価（CPI） | 下落 or 停滞 | 上昇 |
| 企業の業績（大型企業の決算） | 良い | 悪い |
| その他 | 円安、政権の安定 | 円高、不安定な状況、債券売られて長期金利上昇 |

## 地政学リスクの考え方

- 実体経済に影響がなければ → すぐに株高になる
- 実体経済に影響があれば → 株安が長引く
- **長くても米軍介入から30日で相場は上向く**（大和証券データ：1990湾岸戦争、2003イラク戦争、1998イラク空爆、2017シリア空爆、2011リビア空爆、2025イラン空爆、6ケース平均）

## 下落トレンドの投資戦略

正攻法：
1. 投資しない
2. 頑張ってブンゼミパターンを探す
3. 米国株に勝負の場所を移す（分析の仕方は一緒）

番外編：
1. 空売り（分析は買いと逆）
2. 長期投資（ファンダメンタルズ）

## 自動化チェック項目

```
market_score 計算（^N225 の日足ベース）:
  5MA, 10MA, 20MA それぞれの直近10日の傾き (slope)
  + slope > 0 → +1 each
  + slope ≈ 0 → 0 each
  + slope < 0 → -1 each

  合計:
    +3      → "イージー（勝率高）"
    +1〜+2  → "イージー寄りノーマル"
    0       → "ノーマル"
    -1〜-2  → "ノーマル寄りハード"
    -3      → "ハード（勝率低）"
```
