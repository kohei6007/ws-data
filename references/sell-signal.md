# 売りサイン（個別株・スイングトレード）

出典：投資マニュアル §9

## 売却の3段階

### 1次：トレンド周期の警戒
- ローソク足の数（トレンド周期）が **8〜10本** で警戒
- 強いトレンド・テーマ株なら **20本** 程度伸びることもある
- 急上昇銘柄や10%以上の上昇後は **8本目・7本目で利確**してもよい

### 2次：ヨコヨコになるまで待つ
- 陰線続きでヨコヨコになるまで様子見
- **陰線2本連続で下落まで様子見**
- 10MAに下落で反発するかどうか観察
- ※ここまで来てから上がるか下がるかは誰にもわからない

### 3次：明確な売り
- **5MA横ばい or 下向きを陰線で抜ける**
- → 売却

## 補助の売りサイン

- 5MA突破直後にすぐ陰線が出る、またはヨコヨコの場合は一旦売却してもよい
- 再度上昇したら再エントリーでも構わない

## トレンド周期の数え方

- **天辺（土手っぺん）から土底まで**を1トレンドとして数える
- 10本程度が1トレンドの目安
- 日経平均上向き・テーマも強い場合は20本まで伸びることも
- ただし10%以上急上昇している場合は8本目・7本目でも良い
- **複数のトレンドが重なる場合は各トレンドの本数を合計**して総合判断（目安は合計20本に差し掛かるか）。例：11本＋7本＝18本なら「もうすぐ20本＝そろそろ」と判断。相場が弱ければ早めに、強ければ今のトレンド10本目まで待つ判断も可。

## 損切りライン設定（マニュアル §9 後半）

1. 現在の株価から損失5%下の株価を確認（赤色の線で引く）
2. 以下をもとに最終的な損切りラインを確定：
   - **ローソク足**（過去の高値・安値・節目）
   - **移動平均線**
   - **出来高ゾーン**

## 判定アルゴリズム要点（自動化用）

```
sell_state 判定:
  日足5MA下落 + 陰線で5MA下抜け → 「3次：売却」
  陰線2本連続 + 5MA横ばい → 「2次：警戒（売却候補）」
  トレンド周期 ≥ 10 + ヨコヨコ気味 → 「1次：利確検討」
  トレンド周期 ≥ 8 + 急上昇 (+10%以上) → 「1次：早期利確」
  上記以外 → 「ホールド」
```

## 例外（Q&A より）

- 決算は跨がない（決算前までに売却）
- 配当の権利確定日は跨がない（短期トレード前提）
- 年末年始・GW はリスクを考えて売却推奨
- ヨコヨコで損切りラインに達していなくても売却してOK
