# 買いサイン（個別株・スイングトレード）

出典：投資マニュアル §6 / §7

## A. 日足の買いサイン本体

- **A条件: 5MAを陽線で突破**したとき
- **B条件: 最初の5MA近づき**（押し目買いの初動）
  - トレンド周期 **8本以下** が望ましい
  - 5MA・10MA・20MA すべて右肩上がりが理想

### NG条件
- **トレンド周期 10本以上経過してからの5MA近づきはNG**（伸び切っている）
- 日足20MA下向きでの飛びつき買い
- ジリジリ上昇トレンド中の5MA突破飛びつき（→押し目を待つ）

## B. 勝率アップポイント（時間軸別の理想形）

### 月足
- 5MA / 10MA / 20MA / 50MA / 100MA がすべて右肩上がり

### 週足
- 5MA：右肩上がり
- 10MA：右肩上がり（横ばいでもOK）
- 20MA：右肩上がり（横ばいでもOK）
- 50MA：右肩上がり
- 100MA：右肩上がり
- ※週足10MA・20MA横ばいは要注意（伸びにくい）

### 日足
- 5MA：右肩上がり
- 10MA：下落でもよいが**株価が10MA突破している**ほうがよい
- 20MA：右肩上がり
- 50MA：右肩上がり
- 100MA：右肩上がり

## C. 出来高ゾーン（§7）

- 過去3ヶ月の出来高が多い価格帯を「出来高ゾーン」と呼ぶ
- 株価より**上**に出来高ゾーンがある → 上値が重い（買い不利）
- 株価より**下**に出来高ゾーンがある → 下値サポート（買い有利）
- 現在地が出来高ゾーン内 → 反発しやすい

## D. テクニカル分析に向く銘柄（§4）
- 時価総額 5,000億円以上の大型株
- 出来高 10万株以上

## 判定アルゴリズム要点（自動化用）

```
buy_score 計算:
  + 日足5MA陽線突破 (+3)
  + 5MA近づきかつトレンド周期 ≤ 8本 (+2)
  + 日足 5/20MA 右肩上がり (+各1)
  + 週足 5MA 右肩上がり (+2)
  + 週足 10/20MA 右肩上がり or 横ばい (+各1)
  + 月足 5/10/20MA 右肩上がり (+各1)
  − トレンド周期 ≥ 10本での近づき (-3)
  − 日足20MA下向き (-2)
  − 株価上方に出来高ゾーン濃厚 (ユーザー入力時のみ)
```
