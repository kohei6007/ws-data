# 買いサイン（個別株・スイングトレード）

出典：投資マニュアル §6 / §7

## ★大前提：週足・月足を必ず見る（2本目補足で最重要と明言）

- **週足は「絶対」見る**／月足は月1回程度でもよいが必ず確認。
- **週足5MAが右肩上がりは絶対条件**。**週足で買いサイン（5MA突破）が出ていないのに投資してはいけない**（長いトレンドが上向いていない＝勝負しない）。
- 週足20MA右肩上がりが望ましい（少なくとも横ばい）。
- → 日足だけで判断しない。週足が上昇トレンドな銘柄ほど勝率が高い。

## A. 日足の買いサイン本体

- **A条件: 5MAを突破**したとき（終値が5MAの上）
  - **陰線でもOK**（一気に5MAをまたぐ陰線は可）。ただし**終値（株価）が5MAの下にある陰線はNG**＝突破とみなさない。
- **B条件: 最初の5MA近づき**（押し目買いの初動）
  - トレンド周期 **8本以下** が望ましい
  - 5MA・10MA・20MA すべて右肩上がりが理想
- **窓開けの「できたかと思ったローソク足」**＝そこで買った人が多い「ベースキャンプ」。その安値を**下回らない限り上方向**、下回ると上値が重くなると解釈。

### NG条件
- **トレンド周期 10本以上経過してからの5MA近づきはNG**（伸び切っている）
- 日足20MA下向きでの飛びつき買い
- ジリジリ上昇トレンド中の5MA突破飛びつき（→押し目を待つ）
- **日足10MAが右肩下がりのまま5MA突破した場合は、10MAを超えてから入る方が安全**（10MA下向きは弱く、10MAで反発して再下落しやすい）

## B. 勝率アップポイント（時間軸別の理想形）

> **パーフェクトオーダーに近いほど強い**（2本目補足）：日足の 5/10/20/50/100MA が**すべて右肩上がり**の状態が最も勝率が高い。MAの**並び順は多少前後してもよい**（例：50MAが20MAの上でも可）。要点は「各MAが右肩上がり」であること。ただしパーフェクトオーダーでないと投資禁止という意味ではない。
> 週足は日足ほど厳しくなく、**週足20MA・100MAが横ばい**でも良いことがある。

### 月足
- 5MA / 10MA / 20MA / 50MA / 100MA がすべて右肩上がり

### 週足
- 5MA：右肩上がり
- 10MA：右肩上がり（横ばいでもOK）
- 20MA：右肩上がり（横ばいでもOK）
- 50MA：右肩上がり
- 100MA：右肩上がり
- ※週足10MA・20MA横ばいは要注意（伸びにくい）

### 日足
- 5MA：右肩上がり
- 10MA：下落でもよいが**株価が10MA突破している**ほうがよい
- 20MA：右肩上がり
- 50MA：右肩上がり
- 100MA：右肩上がり

## C. 出来高ゾーン（§7）

- 過去3ヶ月の出来高が多い価格帯を「出来高ゾーン」と呼ぶ
- 株価より**上**に出来高ゾーンがある → 上値が重い（買い不利）
- 株価より**下**に出来高ゾーンがある → 下値サポート（買い有利）
- 現在地が出来高ゾーン内 → 反発しやすい

## D. テクニカル分析に向く銘柄（§4）
- 時価総額 5,000億円以上の大型株
- 出来高 10万株以上

## 判定アルゴリズム要点（自動化用）

```
buy_score 計算:
  + 日足5MA陽線突破 (+3)
  + 5MA近づきかつトレンド周期 ≤ 8本 (+2)
  + 日足 5/20MA 右肩上がり (+各1)
  + 週足 5MA 右肩上がり (+2)
  + 週足 10/20MA 右肩上がり or 横ばい (+各1)
  + 月足 5/10/20MA 右肩上がり (+各1)
  − トレンド周期 ≥ 10本での近づき (-3)
  − 日足20MA下向き (-2)
  − 株価上方に出来高ゾーン濃厚 (ユーザー入力時のみ)
```
