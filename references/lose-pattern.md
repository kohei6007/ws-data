# よくある負けパターン

出典：投資マニュアル §8

## パターン1：日足20MA下向きで買い
- 日足の20MAが下向きの状態で買うと勝率悪い
- 「買ってもいいけど、すぐ落ちるかも？」で買うのは負けパターン

## パターン2：ジリジリ上昇中の5MA突破飛びつき
- 20MA右肩上がりでジリジリ上昇トレンド中の5MA突破は注意
- 必要なのは「移動平均線に近づいてきたとき（押し目）」
- → 飛びつかず押しを待つ

## パターン3：週足20MA下向きで無理勝負
- 特に週足で20MA下向きの場合、無理して勝負する必要なし
- 大型相場全体が悪いサインなので見送る

## 自動化チェック項目

```
avoid_flags:
  - 日足20MA 7日連続下落 → "20MA下向きで買い禁止"
  - 直前5本のうち5MA上昇かつ +10% 以上 → "ジリジリ上昇中・飛びつき禁止"
  - 週足20MA 3週連続下落 → "週足20MA下向き・勝負見送り"
```
