# 暴落時の対応

出典：投資マニュアル §13

## 暴落の判定

| レベル | 高値からの下落率 |
|---|---|
| 高値の下落率 | 10〜20% |
| 暴落 | 20〜30% |
| リーマンショック級 | 30〜50% |

| VIX指数 | 名称 |
|---|---|
| 60 | トランプショック級 |
| 80 | 植田ショック級 |

## 暴落の対応

### インデックス投資

**積立投資**
- 指数が高値から **10%以上下落してから増額**（毎日積立を推奨）

**スポット投資**
- 指数が高値から **20%以上下落してから3分割など分割して実施**
- 1度目のスポット投資後に相場が上昇 → それ以降のスポット投資しない
- 1度目のスポット投資後に相場が下落 → 5%〜10%下落するごとに第2弾、第3弾

### 個別株
- **ゼロポジション**で暴落の根本原因と日経平均の5・10MAが上向きになるまで待つ
- 根本原因解決で相場が戻ってきたときの特殊なブンゼミパターンで入る
- 好業績で配当利回り4%以上の銘柄に投資（ファンダメンタルズの知識必要）
- 好業績でこれは絶対に割安と分析できる銘柄に投資（ファンダメンタルズの知識必要）

## 暴落時の買いタイミング（image21）

1. 大底から一度上昇
2. その後に再度5MAを割って下落
3. 大底まで下がらずに再度5MA突破した タイミング
   - = いわゆる「安値切り上がり」の展開
   - 20MAなどが下落中でも気にしない
   - ただし、相場の下落要因がなくなりつつあることを前提とする

## 自動化チェック項目

```
crash_level 判定（^N225 の直近の高値から）:
  -10% 〜 -20% → "下落（積立増額検討）"
  -20% 〜 -30% → "暴落（スポット分割投資検討、個別株はゼロポジ）"
  -30% 〜      → "リーマン級（厳格にファンダメンタル銘柄に絞る）"
  VIX ≥ 60    → "トランプショック級"
  VIX ≥ 80    → "植田ショック級"
```
